1. ER-MIA: Black-Box Adversarial Memory Injection Attacks on Long-Term Memory-Augmented Large Language Models

Source: arXiv:cs.LG | Tags: LLM, Search, RAG, RecSys Deep dive: Full analysis

Summary: A black-box adversarial framework that exploits similarity-based retrieval mechanisms in long-term memory-augmented LLMs by crafting malicious memory entries that are preferentially retrieved over legitimate ones, hijacking model responses without access to model internals. Long-term memory systems introduce a new attack surface beyond the model itself — the retrieval layer — which is largely undefended and exploitable purely through crafted text inputs, even without white-box access to embeddings or model weights. Useful for: Red-teaming / security auditing: Evaluating robustness of RAG pipelines and memory-augmented agents before deployment.

Vault connections: Infinite Retrieval: Attention Enhanced LLMs in Long-Context Processing, context rot, Adversarial Retrieval Noise Training

2. Sparrow: Text-Anchored Window Attention with Visual-Semantic Glimpsing for Speculative Decoding in Video LLMs

Source: arXiv:cs.AI | Tags: LLM, RAG Deep dive: Full analysis

Summary: A speculative decoding framework for Video LLMs that offloads visual computation entirely to the target model by reusing cached text hidden states in a sliding window attention mechanism, while training the draft model on semantically-enriched intermediate layer states instead of raw visual tokens. Standard speculative decoding collapses on Video LLMs because the draft model suffers from (1) attention dilution — visual KV-cache grows proportionally with video length, drowning out relevant signal — and (2) negative visual gain — processing raw visual tokens in the draft model adds noise rather than useful signal, since by deep inference layers the visual semantics have already been absorbed into text hidden states. Useful for: Real-time long-video understanding (25k+ visual tokens) where standard speculative decoding degrades

Vault connections: LLM Inference Performance: The Team Mental Model + Experiment Playbook, Multimodal Textual Knowledge Generation via Open-Vocabulary VLMs, Block Sparse Attention

3. Learning to Configure Agentic AI Systems

Source: HuggingFace | Tags: LLM, RL, Search, RAG Deep dive: Full analysis

Summary: A hierarchical reinforcement learning policy that dynamically selects per-query agent configurations (workflows, tools, token budgets, prompts) rather than applying a single fixed template to all inputs. Static “one-size-fits-all” agent configurations waste compute on easy queries and underperform on hard ones; hand-tuned heuristics are brittle and fail to exploit the combinatorial configuration space effectively. Useful for: Agentic reasoning systems (tool-augmented QA, multi-step planning) where query difficulty varies widely

Vault connections: Synergistic Tool Integration in Agentic Reasoning, Automated Multi-Agent Architecture Search (ADAS, AFlow, EvoFlow), Agentic Reasoning Pipeline

4. NeuroSymActive: Differentiable Neural-Symbolic Reasoning with Active Exploration for Knowledge Graph Question Answering

Source: arXiv:cs.CL | Tags: Embeddings, LLM, Graph, RAG, Search Deep dive: Full analysis

Summary: A modular framework that couples differentiable soft-unification symbolic reasoning with a Monte Carlo value-guided exploration controller to perform efficient multi-hop Knowledge Graph Question Answering (KGQA). Integrating knowledge graph structure with neural models is caught between two failure modes — embedding raw graph facts into prompts is inefficient and brittle, while purely symbolic/search-heavy traversal is expensive in graph lookups and lacks gradient-based refinement. Useful for: Multi-hop question answering over structured KGs (e.g., Freebase, Wikidata, domain ontologies)

Vault connections: KAPING, GraphToken, Multi-source Knowledge Line Graph Prompting (MKLGP)

5. OpaqueToolsBench: Learning Nuances of Tool Behavior Through Interaction

Source: arXiv:cs.CL | Tags: LLM, Search, RAG Deep dive: Full analysis

Summary: Iteratively refines tool documentation by observing execution feedback from tool-calling trajectories, enabling LLM agents to self-improve their understanding of opaque tools without expensive human annotation. Real-world tools are often underspecified—lacking clear best practices, edge case behavior, and failure modes—making existing LLM benchmarks (which assume perfectly documented tools) poor proxies for actual deployment. Useful for: LLM agents operating against third-party or legacy APIs with poor documentation

Vault connections: Synergistic Tool Integration in Agentic Reasoning, Agentic Reasoning Design Patterns, Dynamic Tool Invocation

6. ScrapeGraphAI-100k: A Large-Scale Dataset for LLM-Based Web Information Extraction

Source: arXiv:cs.IR | Tags: LLM, Search, RAG Deep dive: Full analysis

Summary: A real-world, large-scale dataset of LLM-based structured web extraction events is constructed from opt-in telemetry, then deduplicated and schema-balanced to produce high-quality training and benchmark examples. Existing web information extraction datasets are small, synthetic, or text-only, lacking structural context (schemas, JSON outputs, validation metadata) needed to train or benchmark LLMs on realistic structured extraction tasks. Useful for: Fine-tuning small language models (1–3B) for structured extraction, narrowing the gap to much larger models (30B+)

Vault connections: Millions of GeAR: Extending GraphRAG to Millions of Document, Multi-Partite Graph Index, Graph-Enhanced RAG

7. On Surprising Effectiveness of Masking Updates in Adaptive Optimizers

Source: arXiv:cs.AI | Tags: LLM Deep dive: Full analysis

Summary: Selectively mask parameter updates based on alignment between current gradients and momentum, inducing curvature-aware geometric regularization that smooths optimization trajectories. Dense adaptive optimizers (Adam, Muon) apply updates to all parameters at every step, accumulating noise along high-curvature directions and making optimization trajectories erratic; existing methods address this with increasingly complex preconditioners at significant computational cost. Useful for: LLM pre-training at scale (demonstrated 1B+ parameter models with >19% perplexity reduction vs. Adam)

Vault connections: Large Language Models as Optimizers, Saliency-Aware Pruning with Learnable Masks, Adaptive Low-Precision Training with Mixed Precision Caching

8. Recursive Concept Evolution for Compositional Reasoning in Large Language Models

Source: arXiv:cs.CL | Tags: LLM, RL, Search, RAG Deep dive: Full analysis

Summary: RCE enables pretrained LLMs to dynamically construct new low-rank representational subspaces during inference when existing latent geometry is insufficient to support a required abstraction. LLMs hit a hard ceiling on compositional reasoning tasks (ARC-AGI-2, GPQA, MATH, BBH, HLE) because their representation space is fixed at inference time — when a required abstraction isn’t already encoded, the model cannot construct it, causing sharp accuracy degradation that token-level search methods (chain-of-thought, self-consistency, RL) cannot fix. Useful for: Compositional and multi-step reasoning tasks that require building new abstractions from primitives (ARC-AGI-2, math olympiad problems)

Vault connections: Reasoning Beyond Limits: Advances and Open Problems for LLMs, Compositional Reasoning Retrieval Evaluation, Take a Step Back: Evoking Reasoning via Abstraction in Large Language Model

9. ResearchGym: Evaluating Language Model Agents on Real-World AI Research

Source: arXiv:cs.AI | Tags: LLM, Search, RAG Deep dive: Full analysis

Summary: A containerized benchmark environment that evaluates AI agents on end-to-end scientific research by providing baseline implementations from real published papers while withholding the proposed methods, requiring agents to independently discover and surpass state-of-the-art solutions. Existing agent benchmarks test narrow, synthetic tasks rather than the full research loop (hypothesis generation → experimentation → evaluation), making it impossible to measure whether agents can perform genuine open-ended scientific discovery in realistic computational environments. Useful for: Benchmarking frontier AI agents on realistic long-horizon scientific tasks

Vault connections: Taxonomy-Guided Benchmarking for Agentic Evaluation, Agentic Reasoning Design Patterns, Language Agents by Yu Su

10. Detecting Overflow in Compressed Token Representations for Retrieval-Augmented Generation

Source: HuggingFace | Tags: LLM, Search, RAG Deep dive: Full analysis

Summary: Train lightweight probing classifiers over joint query and compressed-context representations to detect when soft token compression has discarded information necessary to answer a query, enabling pre-LLM gating before inference. Soft compression architectures (e.g., xRAG) reduce token sequences to compact learned representations, but provide no signal about whether a given query’s answer survives compression — leading to silent failures where the LLM confidently generates wrong answers from information-lossy inputs. Useful for: RAG pipelines with soft compression where latency savings must be balanced against answer reliability

Vault connections: Infinite Retrieval: Attention Enhanced LLMs in Long-Context Processing, Context Compression, Lengthy Context Retrieval via LLM Pretraining

1. 1-Bit Wonder: Improving QAT Performance in the Low-Bit Regime through K-Means Quantization

Summary: Replace uniform integer quantization grids with k-means-learned codebooks during quantization-aware training, then show that 1-bit k-means quantization maximizes downstream generative task performance under a fixed inference memory budget. Standard QAT uses integer (uniform) quantization formats whose fixed grid spacing is suboptimal for the non-uniform distributions of neural network weights; additionally, the QAT design space (format × bit-width) is underexplored, and evaluations rely on perplexity rather than generative task performance, obscuring the true quality/memory trade-off. Useful for: Deploying large language models on memory-constrained hardware (edge devices, consumer GPUs) where integer quantization leaves accuracy on the table

Deep dive: Full analysis

2. COMPOT: Calibration-Optimized Matrix Procrustes Orthogonalization for Transformers Compression

Summary: Replace truncated SVD with sparse dictionary learning using orthogonal dictionaries that admit closed-form Procrustes solutions and single-step sparse coding, enabling training-free Transformer weight compression without iterative optimization. Standard low-rank SVD compression forces all weight structure into a single shared subspace, degrading accuracy at moderate compression ratios. Useful for: Compressing large language models (LLMs) for inference on memory-constrained hardware without fine-tuning

Deep dive: Full analysis

3. Enabling Low-Latency Machine learning on Radiation-Hard FPGAs with hls4ml

Summary: A new synthesis backend for the hls4ml library that automatically translates trained neural network models into High-Level Synthesis (HLS) projects targeting Microchip PolarFire radiation-hard FPGAs, enabling on-detector machine learning inference in high-radiation environments. The hls4ml library — the standard tool in high-energy physics (HEP) for deploying ML on FPGAs — had no support for radiation-hard FPGAs (e.g., PolarFire), blocking adoption of on-detector ML in experiments where commercial Xilinx/Intel FPGAs would fail due to radiation-induced bit flips and latch-up events. Useful for: High-energy physics trigger systems requiring sub-100 ns inference latency at LHC detector frontends (e.g., LHCb, ATLAS, CMS upgrade detectors)

5. The Equalizer: Introducing Shape-Gain Decomposition in Neural Audio Codecs

Summary: Decompose audio signals into a normalized shape vector and a scalar gain before encoding, process them through separate quantization paths, and reconstruct the output signal post-decoding — preventing gain variation from polluting the NAC latent space. Neural Audio Codecs (NACs) encode gain (short-term energy) and shape (normalized structure) jointly in the same latent space, causing global amplitude variations to strongly perturb encoder embeddings and their vector quantization, wasting codebook capacity on level-dependent redundancy and degrading bitrate-distortion performance. Useful for: Neural speech/audio compression at low-to-mid bitrates where codebook efficiency is critical

Deep dive: Full analysis

6. The Vision Wormhole: Latent-Space Communication in Heterogeneous Multi-Agent Systems

Summary: Repurpose the visual input pathway of Vision-Language Models (VLMs) as a universal inter-agent communication channel by encoding heterogeneous reasoning traces into synthetic visual tokens, enabling text-free, model-agnostic latent state transfer between agents of different architectures. Standard multi-agent LLM systems communicate via serialized text, incurring tokenization overhead, context-length bottlenecks, and lossy information quantization. Useful for: Heterogeneous multi-agent pipelines where different tasks are routed to specialized model families (e.g., Qwen-VL for vision-heavy subtasks, Gemma for text reasoning)

Deep dive: Full analysis

7. jina-embeddings-v5-text: Task-Targeted Embedding Distillation

Summary: Combine knowledge distillation from a large teacher embedding model with task-specific contrastive loss during student training, jointly optimizing for both teacher alignment and discriminative task performance. Small embedding models trained purely with contrastive loss lack the representational richness of larger models, while pure distillation ignores task-specific discriminative structure — TTED addresses the quality/size tradeoff by leveraging both signals simultaneously. Useful for: Information retrieval / dense passage retrieval where compact models are needed at query-time
