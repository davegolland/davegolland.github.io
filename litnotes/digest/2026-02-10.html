1. LLaDA2.1: Speeding Up Text Diffusion via Token Editing

Source: arXiv:cs.AI | Tags: RL, Diffusion, LLM Deep dive: Full analysis

Summary: Combine Mask-to-Token (M2T) prediction with Token-to-Token (T2T) editing in a configurable threshold-based decoding scheme to enable speed-quality tradeoffs in diffusion language models. Traditional diffusion language models face a rigid tradeoff between decoding speed and generation quality. Useful for: Code generation tasks requiring high throughput (achieves 663-892 TPS on coding benchmarks)

2. OSCAR: Optimization-Steered Agentic Planning for Composed Image Retrieval

Source: arXiv:cs.AI | Tags: Search, Embeddings, RAG Deep dive: Full analysis

Summary: Reformulates agentic composed image retrieval from heuristic trial-and-error into a principled trajectory optimization problem, using offline mixed-integer programming to derive optimal retrieval trajectories that are stored as in-context demonstrations for online VLM planner steering. Addresses the limitations of both unified embedding retrieval (single-model myopia) and heuristic agentic retrieval (suboptimal trial-and-error orchestration) by replacing ad-hoc exploration with mathematically optimal planning trajectories. Useful for: Composed image retrieval tasks requiring multi-constraint reasoning (visual + textual modifications)

Vault connections: HM-RAG: Hierarchical Multi-Agent Multimodal Retrieval Augmented Generation, Modular Plug-and-Play Multi-Source Retrieval Agents, Multi-Agent Agentic RAG

3. Do MLLMs Really See It: Reinforcing Visual Attention in Multimodal LLMs

Source: arXiv:cs.AI | Tags: RL, LLM, RAG Deep dive: Full analysis

Summary: A reinforcement learning framework that trains multimodal LLMs with explicit region-level visual attention rewards to learn stable visual grounding policies during chain-of-thought reasoning. Current MLLMs exhibit weak visual attention during reasoning—early-stage visual misalignment propagates through reasoning chains without correction because training lacks proper credit assignment for where the model looks, leading to failed inferences even with correct reasoning structure. Useful for: Complex visual reasoning tasks requiring multi-hop inference over image regions

Vault connections: Reasoning Beyond Limits: Advances and Open Problems for LLMs, Multimodal Textual Knowledge Generation via Open-Vocabulary VLMs, Multimodal Knowledge Graph Construction with Contextualized Visual Entities

4. Improving Data and Reward Design for Scientific Reasoning in Large Language Models

Source: arXiv:cs.CL | Tags: RL, RAG, LLM Deep dive: Full analysis

Summary: A three-stage scientific reasoning training pipeline that combines exploration-expanding supervised fine-tuning, dynamic difficulty curriculum, and rubric-guided reinforcement learning to enable stable RL on open-ended scientific questions. Standard SFT → RL workflows fail on scientific reasoning due to: (1) narrow reasoning pattern coverage before RL, (2) static training data mismatched to evolving model capability, and (3) inability to provide stable reward signals for open-ended scientific answers without verifiable ground truth. Useful for: Training models for scientific question answering (chemistry, physics, biology, mathematics)

Vault connections: Reasoning in Language Models - Hanna Hajishirzi, Mathematical Problem-Solving Synthetic Query Generation, Beyond Chunks and Graphs: Retrieval-Augmented Generation through Triplet-Driven Thinking

5. LIT-GRAPH: Evaluating Deep vs. Shallow Graph Embeddings for High-Quality Text Recommendation in Domain-Specific Knowledge Graphs

Source: arXiv:cs.IR | Tags: RecSys, Graph, RAG, LLM, Embeddings Deep dive: Full analysis

Summary: A domain-specific knowledge graph recommendation system that compares shallow (DeepWalk, Biased Random Walk) and deep (R-GCN) graph embeddings to balance structural connectivity and semantic relevance for pedagogically aligned literature recommendations. Addresses curriculum stagnation in English education by scaffolding teachers to discover diverse, pedagogically appropriate literature beyond commonly taught texts, while navigating the trade-off between graph structure fidelity and domain-specific semantic quality in recommendations. Useful for: Educational content recommendation: Curriculum design, course material selection, adaptive learning paths

Vault connections: Instance-Aware Knowledge Distillation from LLMs to Sequential Models (DLLM2Rec), Knowledge Graph Retrieval-Augmented Generation for LLM-based Recommendation, A novel recommender system using light graph convolutional network and personalized knowledge-aware attention sub-network

3. Rethinking Generative Recommender Tokenizer: Recsys-Native Encoding and Semantic Quantization Beyond LLMs

Summary: A generative recommendation framework that learns collaborative-predictive item representations and quantizes them into semantic IDs optimized for sequential predictability, without relying on pre-trained language models. Existing semantic ID methods use foundation model embeddings weakly coupled with collaborative signals and generic quantization schemes (like RQ-VAE) that don’t minimize sequential uncertainty for autoregressive generation, leading to suboptimal recommendation quality and inefficient tokenization. Useful for: Large-scale sequential recommendation systems (e-commerce, video, music streaming)

4. ViCA: Efficient Multimodal LLMs with Vision-Only Cross-Attention

Summary: A sparse multimodal architecture where visual tokens bypass all self-attention and feed-forward layers in the language model, interacting with text only through cross-attention at a small subset of strategically selected layers. Existing multimodal LLMs (MLLMs) process visual tokens through every Transformer layer alongside text tokens, causing massive computational overhead despite visual embeddings already being well-aligned with the language space after projection. Useful for: Vision-language tasks requiring efficient inference: VQA, image captioning, visual reasoning

Deep dive: Full analysis

5. Predicting Future Utility: Global Combinatorial Optimization for Task-Agnostic KV Cache Eviction

Summary: Optimize KV cache eviction by allocating per-head budgets based on marginal utility for preserving long-term semantic information, rather than treating all attention heads uniformly with instantaneous heuristic metrics. Existing KV cache eviction methods assume score magnitudes are consistent proxies for importance across all attention heads, ignoring that different heads have heterogeneous predictive fidelity—some prioritize instantaneous token contribution while others capture long-horizon utility. Useful for: Long-context LLM inference (document QA, summarization, multi-turn dialogue)

Deep dive: Full analysis

6. ParisKV: Fast and Drift-Robust KV-Cache Retrieval for Long-Context LLMs

Summary: A drift-robust, GPU-native KV-cache retrieval framework that uses collision-based candidate selection followed by quantized inner-product reranking to enable efficient top-k attention on long contexts, with CPU-offloading support via Unified Virtual Addressing for million-token scale. Existing KV-cache retrieval methods for long-context LLMs suffer from distribution drift between prefill and decode phases, high retrieval latency at scale, and inability to handle million-token contexts due to memory constraints. Useful for: Long-document understanding and retrieval (RAG over 100k+ token contexts)

Deep dive: Full analysis

7. Attn-GS: Attention-Guided Context Compression for Efficient Personalized LLMs

Summary: Use attention patterns from a fine-tuned marking model to identify and preserve important personalization signals, then guide a compression model to generate condensed user contexts that retain task-relevant information. Existing context compression methods use heuristic selection (e.g., recency) or blind summarization that treats user profiles as monolithic text, failing to leverage how LLMs internally prioritize information during personalization tasks, resulting in poor quality-compression tradeoffs. Useful for: Personalized chatbots with extensive user interaction histories

8. Semantic Search At LinkedIn

Summary: A co-designed inference system that treats ranking as a prefill-dominated workload, using context compression and hybrid text-embedding interactions to enable 75x throughput improvement for production LLM-based semantic search. Traditional LLM ranking suffers from poor throughput (10-100x slower than classical models) due to sequential token generation and large context windows when scoring multiple candidates, making it infeasible for high-QPS production retrieval systems. Useful for: Production semantic search with strict latency SLAs (job search, people search, product retrieval)

Deep dive: Full analysis

9. ManifoldKV: Training-Free KV Cache Compression via Euclidean Outlier Detection

Summary: Score KV cache tokens for eviction using Euclidean distance to the key centroid rather than cosine similarity, preserving both angular direction and magnitude information to identify semantically salient outliers. Cosine-similarity-based eviction methods are scale-invariant and discard magnitude cues, causing directional collisions in multi-key retrieval and centroid dilution at extreme context lengths (64K+), leading to catastrophic performance collapse. Useful for: Long-context LLM inference (4K-128K tokens) with memory constraints

Deep dive: Full analysis

10. DynamiQ: Accelerating Gradient Synchronization using Compressed Multi-hop All-reduce

Summary: A gradient quantization framework optimized for multi-hop all-reduce topologies that dynamically re-quantizes partial sums at intermediate aggregation hops using adaptive scaling and range tracking, paired with a fused decompress-accumulate-recompress kernel. Existing gradient quantization methods (e.g., THC, Omni-Reduce, MXFP) are designed for single-hop aggregation and suffer significant accuracy degradation in multi-hop topologies because they don’t account for partial sums accumulating quantization errors across multiple hops. Useful for: Large-scale distributed training (1000+ GPUs) with multi-tier network topologies (rack → pod → cluster aggregation)

Deep dive: Full analysis

11. Generative Reasoning Re-ranker

Summary: A three-stage LLM training framework (semantic ID mid-training → reasoning trace distillation → RL with verifiable rewards) that teaches models to rerank recommendations through explicit chain-of-thought reasoning rather than implicit scoring. - Existing LLM-based recommenders neglect the critical reranking phase (focusing only on retrieval/ranking)

  • Standard LLM approaches underutilize reasoning capabilities enhanced through RL and high-quality reasoning data
  • Non-semantic item IDs create severe scalability issues in industrial systems with billions of items
  • Models exploit reward hacking by preserving input order rather than truly optimizing reranking quality Useful for: Industrial recommendation reranking with massive item catalogs (billions of items)

12. OJBKQ: Objective-Joint Babai-Klein Quantization

Summary: A layer-wise post-training quantization method that jointly optimizes over activations and weights by formulating each layer’s quantization as a box-constrained integer least squares problem, then solving it approximately using extended Babai and Klein algorithms with K-best sampling. Existing weight-only post-training quantization methods rely on heuristic objectives and greedy rounding, causing significant accuracy degradation at low bit-widths (3-4 bits). Useful for: Post-training quantization of large language models for deployment on resource-constrained devices
