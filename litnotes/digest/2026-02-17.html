1. Can I Have Your Order? Monte-Carlo Tree Search for Slot Filling Ordering in Diffusion Language Models

Source: HuggingFace | Tags: Diffusion, Search, RAG, LLM Deep dive: Full analysis

Summary: Formulate slot infilling order selection in Masked Diffusion Models (MDMs) as a sequential decision-making problem, optimized via Monte Carlo Tree Search (MCTS) with look-ahead simulations to discover orderings that maximize generation quality. Plan-and-infill decoding in MDMs is highly sensitive to the order in which masked slots are filled — different orderings produce high output variance, and naive or sequential heuristics leave significant performance on the table. Useful for: Code generation tasks where sub-routine dependencies make ordering critical (e.g., MBPP benchmarks)

Vault connections: Monte Carlo Tree Search, Context Slotting, Block Sparse Attention

2. SPILLage: Agentic Oversharing on the Web

Source: HuggingFace | Tags: RecSys, RAG, LLM Code: https://github.com/jrohsc/SPILLage Deep dive: Full analysis

Summary: A two-dimensional taxonomy (channel × directness) that formally characterizes how LLM web agents unintentionally disclose task-irrelevant user information through both textual outputs and observable behavioral traces. Prior privacy work on LLM agents focused exclusively on text leakage (what agents type), missing a larger and more insidious class of privacy violation: behavioral signals (clicks, scrolls, navigation) that third parties can monitor to infer sensitive user attributes even when no private text is transmitted. Useful for: Auditing LLM agent deployments that access personal data (email, calendar, health) before acting on the open web

Vault connections: A Survey on LLM-powered Agents for Recommender Systems, The Future is Agentic: Definitions, Perspective, and Open Challenge of Multi-Agent Recommender System, Agentic Reasoning: Main Takeaways and Techniques

3. DHPLT: large-scale multilingual diachronic corpora and word representations for semantic change modelling

Source: HuggingFace | Tags: RAG, Search, Embeddings, LLM Code: https://data.hplt-project.org/three/diachronic/ Deep dive: Full analysis

Summary: A large-scale multilingual diachronic corpus collection spanning 41 languages and three time periods, built from web-crawled data with timestamp-based temporal partitioning, paired with pre-computed word embeddings and lexical substitutions for semantic change modelling. The scarcity of multilingual diachronic corpora for semantic change detection — most existing resources cover only a handful of high-resource languages, blocking cross-lingual and typologically diverse studies of how word meanings shift over time. Useful for: Cross-lingual semantic change detection and benchmarking

Vault connections: Infinite Retrieval: Attention Enhanced LLMs in Long-Context Processing, Dense Passage Retrieval in Conversational Search, DocMMIR Cross-Domain Multimodal Benchmark Construction
