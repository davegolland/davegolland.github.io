Algorithmic Prompt-Augmentation for Efficient LLM-Based Heuristic Design for A* Search

Source: arXiv:cs.AI Published: 2026-01-27 Rank: #7 (score: 0.97, max_sim: 0.83) Reading time: ~15 min Tags: Search, RAG, LLM

Abstract

Heuristic functions are essential to the performance of tree search algorithms such as A*, where their accuracy and efficiency directly impact search outcomes. Traditionally, such heuristics are handcrafted, requiring significant expertise. Recent advances in large language models (LLMs) and evolutionary frameworks have opened the door to automating heuristic design. In this paper, we extend the Evolution of Heuristics (EoH) framework to investigate the automated generation of guiding heuristics for A* search. We introduce a novel domain-agnostic prompt augmentation strategy that includes the A* code into the prompt to leverage in-context learning, named Algorithmic - Contextual EoH (A-CEoH). To evaluate the effectiveness of A-CeoH, we study two problem domains: the Unit-Load Pre-Marshalling Problem (UPMP), a niche problem from warehouse logistics, and the classical sliding puzzle problem (SPP). Our computational experiments show that A-CEoH can significantly improve the quality of the generated heuristics and even outperform expert-designed heuristics.

Key Techniques

  • Evolution of Heuristics (EoH) framework
  • prompt augmentation strategy with A* code inclusion
  • in-context learning for heuristic generation

Analysis: Algorithmic Prompt-Augmentation for A* Heuristic Design

Extending LLM-as-Optimizer Frameworks

This paper directly extends the paradigm established in “Large Language Models as Optimizers,” which demonstrated that LLMs can act as derivative-free optimizers through natural language prompting. Where OPRO applies this concept to mathematical optimization problems like linear regression and TSP, A-CEoH applies it to the specialized domain of search heuristic design. The key innovation—incorporating the A* algorithm code directly into the prompt as “contextual knowledge”—represents a domain-specific refinement of OPRO’s in-context learning approach. Rather than having the LLM optimize prompts in isolation, A-CEoH grounds the optimization process in the algorithmic context of the actual search mechanism being optimized, leveraging the LLM’s ability to understand and reason about code structures.

Algorithmic Contextualization as Prompt Evolution Strategy

The paper’s prompt augmentation strategy aligns with the “Reflective Prompt Evolution” concepts discussed in GEPA Explained, where iterative refinement and evolution of prompts themselves become the optimization target. A-CEoH implements this through the Evolution of Heuristics framework, systematically evolving heuristic functions rather than static prompts. However, it introduces a crucial innovation beyond standard APE: by embedding algorithmic context (the A* implementation), it enables what could be termed “algorithm-aware prompt optimization.” This bridges the gap between generic prompt optimization and domain-specific algorithm design—the LLM isn’t just generating better prompts or heuristics, but doing so with explicit understanding of the computational structure that will evaluate them.

Novel Contribution: Context-Grounded Evolutionary Optimization

Where prior work in Automatic Prompt Optimization focused on optimizing prompts for downstream NLP tasks, A-CEoH demonstrates that this approach can be applied to generate algorithmic components themselves. The domain-agnostic nature of the prompt augmentation strategy—validated on both warehouse logistics (UPMP) and classical puzzles (SPP)—suggests that providing algorithmic context is a generalizable technique for improving LLM-based optimization across search domains. The finding that A-CEoH can outperform expert-designed heuristics represents a significant extension of the OPRO thesis: not only can LLMs optimize through prompting, but they can do so to match or exceed human expertise in specialized algorithmic domains when given proper contextual grounding.
