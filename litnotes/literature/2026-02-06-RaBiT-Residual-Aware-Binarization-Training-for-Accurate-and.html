RaBiT: Residual-Aware Binarization Training for Accurate and Efficient LLMs

Source: arXiv:cs.AI Published: 2026-02-05 Rank: #8 (score: 0.99, max_sim: 0.82) Reading time: ~20 min Tags: Embeddings, LLM, Early-Exit, NSA

Abstract

Efficient deployment of large language models (LLMs) requires extreme quantization, forcing a critical trade-off between low-bit efficiency and performance. Residual binarization enables hardware-friendly, matmul-free inference by stacking binary (1) layers, but is plagued by pathological feature co-adaptation. We identify a key failure mode, which we term inter-path adaptation: during quantization-aware training (QAT), parallel residual binary paths learn redundant features, degrading the error-compensation structure and limiting the expressive capacity of the model. While prior work relies on heuristic workarounds (e.g., path freezing) that constrain the solution space, we propose RaBiT, a novel quantization framework that resolves co-adaptation by algorithmically enforcing a residual hierarchy. Its core mechanism sequentially derives each binary path from a single shared full-precision weight, which ensures that every path corrects the error of the preceding one. This process is stabilized by a robust initialization that prioritizes functional preservation over mere weight approximation. RaBiT redefines the 2-bit accuracy-efficiency frontier: it achieves state-of-the-art performance, rivals even hardware-intensive Vector Quantization (VQ) methods, and delivers a inference speed-up over full-precision models on an RTX 4090.

Key Techniques

  • Residual binarization
  • inter-path adaptation mitigation
  • sequential binary path derivation

RaBiT: Residual-Aware Binarization Training

Core Idea: A quantization framework that enforces a residual hierarchy by sequentially deriving binary paths from a single shared full-precision weight, ensuring each path explicitly corrects the errors of the preceding path rather than learning redundant features.

Problem It Solves: Addresses pathological inter-path adaptation in residual binarization during quantization-aware training (QAT), where parallel binary paths co-adapt and learn redundant features instead of maintaining an error-compensation structure, limiting model expressiveness and degrading accuracy.

How It Works:

  1. Maintain a single shared full-precision weight matrix as the source of truth
  2. Sequentially derive each binary (1) path from this shared weight, with each path explicitly computed to minimize the residual error left by the previous paths

Where It’s Useful:

  • Extreme quantization (2-bit) for LLM deployment on resource-constrained devices
  • Hardware-accelerated inference with binary operations (replacing expensive matmuls with additions)
  • Edge deployment scenarios requiring 4-5× inference speedups with minimal accuracy degradation

Limitations / Caveats:

  • Requires quantization-aware training (QAT) rather than post-training quantization (PTQ), increasing training overhead
  • Sequential path derivation may increase training complexity compared to naive parallel path training
  • Performance benefits are hardware-dependent and maximize on systems optimized for binary operations

  • Enable on-device LLM inference for personalized recommendations without cloud round-trips
  • Compress embedding models for billion-scale retrieval where memory bandwidth is critical, combining with structured pruning for compound efficiency gains
