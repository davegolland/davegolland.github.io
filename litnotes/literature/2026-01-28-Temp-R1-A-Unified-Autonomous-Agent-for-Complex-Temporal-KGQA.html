Temp-R1: A Unified Autonomous Agent for Complex Temporal KGQA via Reverse Curriculum Reinforcement Learning

Source: arXiv:cs.CL Published: 2026-01-26 Rank: #6 (score: 0.99, max_sim: 0.83) Reading time: ~15 min Tags: RAG, LLM, Graph, RL Code: https://github.com/zjukg/Temp-R1

Abstract

Temporal Knowledge Graph Question Answering (TKGQA) is inherently challenging, as it requires sophisticated reasoning over dynamic facts with multi-hop dependencies and complex temporal constraints. Existing methods rely on fixed workflows and expensive closed-source APIs, limiting flexibility and scalability. We propose Temp-R1, the first autonomous end-to-end agent for TKGQA trained through reinforcement learning. To address cognitive overload in single-action reasoning, we expand the action space with specialized internal actions alongside external action. To prevent shortcut learning on simple questions, we introduce reverse curriculum learning that trains on difficult questions first, forcing the development of sophisticated reasoning before transferring to easier cases. Our 8B-parameter Temp-R1 achieves state-of-the-art performance on MultiTQ and TimelineKGQA, improving 19.8% over strong baselines on complex questions. Our work establishes a new paradigm for autonomous temporal reasoning agents. Our code will be publicly available soon at https://github.com/zjukg/Temp-R1.

Analysis: Temp-R1 and the Evolution of Agentic Reasoning Systems

Building on Graph-R1’s Multi-Step Agent Framework: Temp-R1 extends the agentic reasoning paradigm established by Graph-R1 but with a critical shift in problem domain and action design. While Graph-R1 operates within a knowledge hypergraph environment performing “think-retrieve-rethink-generate” loops, Temp-R1 introduces specialized internal actions alongside external actions to address temporal reasoning complexity. This expansion of the action space is a response to a different cognitive challenge—not just retrieval efficiency and reasoning quality (Graph-R1’s focus), but preventing the agent from taking shortcuts on simple questions while maintaining sophistication on complex temporal dependencies. Both papers share the core insight that autonomous agents need structured, iterative reasoning loops, but Temp-R1’s reverse curriculum learning represents a novel training methodology that Graph-R1 doesn’t address: forcing the agent to learn complex patterns first prevents overfitting to simple question-solving strategies.

Temporal Reasoning Requires Explicit Memory Architecture: Temp-R1’s internal actions echo the persistent memory and logic tracking problem that the Mind-Map Agent solves. Just as the Mind-Map Agent addresses “loss of coherence in long reasoning chains” through dynamic knowledge graph construction, Temp-R1’s internal actions likely function as temporal constraint tracking and multi-hop dependency management—making explicit what a naive agent might lose across reasoning steps. However, Temp-R1 appears to integrate this memory directly into the action space rather than as a parallel graph-construction process, suggesting a tighter coupling between reasoning and memory management for temporal questions specifically.

New Paradigm: Reverse Curriculum as a Training Inversion: The most novel contribution Temp-R1 adds to this knowledge base is the reverse curriculum learning framework. While “Reasoning Beyond Limits” surveys recent advances in inference-time scaling and RL-based reasoning (DeepSeek-R1, o1, etc.), it doesn’t address training-time curriculum design for preventing shortcut learning. Temp-R1’s insight—that training on difficult temporal questions first forces development of sophisticated reasoning before transfer to easier cases—inverts traditional curriculum learning assumptions and directly targets a failure mode in knowledge-intensive reasoning: models learning surface-level patterns rather than deep temporal logic. This suggests that specialized RL agents for constrained domains (like temporal QA) benefit from problem-domain-specific curriculum strategies beyond generic inference-time scaling.
