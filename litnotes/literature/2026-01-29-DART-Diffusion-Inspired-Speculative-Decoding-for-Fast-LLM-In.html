DART: Diffusion-Inspired Speculative Decoding for Fast LLM Inference

DART: Diffusion-Inspired Speculative Decoding for Fast LLM Inference

Source: arXiv:cs.CL Published: 2026-01-27 Rank: #5 (score: 0.92, max_sim: 0.87) Reading time: ~0 min Tags: Uncategorized

Abstract

Speculative decoding is an effective and lossless approach for accelerating LLM inference. However, existing widely adopted model-based draft designs, such as EAGLE3, improve accuracy at the cost of multi-step autoregressive inference, resulting in high drafting latency and ultimately rendering the drafting stage itself a performance bottleneck. Inspired by diffusion-based large language models (dLLMs), we propose DART, which leverages parallel generation to reduce drafting latency. DART predicts logits for multiple future masked positions in parallel within a single forward pass based on hidden states of the target model, thereby eliminating autoregressive rollouts in the draft model while preserving a lightweight design. Based on these parallel logit predictions, we further introduce an efficient tree pruning algorithm that constructs high-quality draft token trees with N-gram-enforced semantic continuity. DART substantially reduces draft-stage overhead while preserving high draft accuracy, leading to significantly improved end-to-end decoding speed. Experimental results demonstrate that DART achieves a 2.03x—3.44x wall-clock time speedup across multiple datasets, surpassing EAGLE3 by 30% on average and offering a practical speculative decoding framework. Code is released at https://github.com/fvliang/DART.

Key Techniques

  • Parallel logit prediction
  • Tree pruning algorithm
  • N-gram-enforced semantic continuity
  • Diffusion-based draft generation

Analysis: DART and Connections to Prior Speculative Decoding Work

DART extends the speculative decoding paradigm documented in the matched notes by addressing a critical bottleneck that previous approaches like EAGLE3 introduced: drafting latency from multi-step autoregressive inference. While the matched notes discuss optimization of draft shapes (Draft Shape Optimization for Dense Speculative Decoding) and batch verification strategies (Batch Verification in Speculative Decoding), DART tackles the fundamental architectural inefficiency of sequential token generation during the draft stage itself. By drawing inspiration from diffusion-based language models, DART predicts logits for multiple future masked positions in parallel within a single forward pass, eliminating the autoregressive rollout entirely. This represents a shift from optimizing how many drafts or how long each draft should be (as explored in DReSD’s 3×20 vs n×k analysis) to fundamentally changing how the drafts are generated—moving from sequential to parallel prediction. The 30% average speedup over EAGLE3 suggests that drafting latency had indeed become the bottleneck that the “LLM Inference Performance” mental model would predict: even with good draft acceptance rates, if the draft generation itself is slow, end-to-end performance suffers.

The paper’s N-gram-enforced tree pruning algorithm bridges DART’s parallel logit predictions to the practical concerns raised in the vault notes about draft quality and verification efficiency. The matched notes emphasize that batch verification (processing multiple draft sequences in one LLM forward pass) is key to reducing latency, and that draft shape matters for balancing verification cost against acceptance rates. DART’s tree construction approach appears to synthesize these insights: by generating multiple token predictions in parallel and then pruning them into high-quality trees with semantic continuity constraints, it creates a draft structure optimized for batch verification while maintaining the lightweight design principle. The 2.03×–3.44× wall-clock speedup across datasets suggests DART successfully reduces the per-token decode cost identified in the “Team Mental Model” by shrinking both the drafting overhead and maintaining favorable KV-cache behavior—since accepted speculative tokens avoid the serial decode loop bottleneck.

What DART adds to the knowledge base is a hybrid architecture that combines parallel prediction (diffusion-inspired) with tree-based speculation, moving beyond the binary choice between retrieval-based drafts (DReSD) and autoregressive model-based drafts (EAGLE). This opens new design space: instead of optimizing draft shape for a fixed generation method, DART optimizes the generation method itself to be parallelizable, then constructs optimal trees from those predictions. The practical implication aligns with the “Experiment Playbook” philosophy in the matched notes—DART demonstrates that rearchitecting the draft stage to exploit parallelism yields larger gains than further tuning autoregressive draft models. This suggests future speculative decoding research should prioritize parallel prediction mechanisms over incremental improvements to sequential draft models, especially as context lengths grow and KV-cache bandwidth becomes increasingly constrained during decode.
