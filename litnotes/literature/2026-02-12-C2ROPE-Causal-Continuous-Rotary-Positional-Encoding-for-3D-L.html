C^2ROPE: Causal Continuous Rotary Positional Encoding for 3D Large Multimodal-Models Reasoning

Abstract

Recent advances in 3D Large Multimodal Models (LMMs) built on Large Language Models (LLMs) have established the alignment of 3D visual features with LLM representations as the dominant paradigm. However, the inherited Rotary Position Embedding (RoPE) introduces limitations for multimodal processing. Specifically, applying 1D temporal positional indices disrupts the continuity of visual features along the column dimension, resulting in spatial locality loss. Moreover, RoPE follows the prior that temporally closer image tokens are more causally related, leading to long-term decay in attention allocation and causing the model to progressively neglect earlier visual tokens as the sequence length increases. To address these issues, we propose C^2RoPE, an improved RoPE that explicitly models local spatial Continuity and spatial Causal relationships for visual processing. C^2RoPE introduces a spatio-temporal continuous positional embedding mechanism for visual tokens. It first integrates 1D temporal positions with Cartesian-based spatial coordinates to construct a triplet hybrid positional index, and then employs a frequency allocation strategy to encode spatio-temporal positional information across the three index components. Additionally, we introduce Chebyshev Causal Masking, which determines causal dependencies by computing the Chebyshev distance of image tokens in 2D space. Evaluation results across various benchmarks, including 3D scene reasoning and 3D visual question answering, demonstrate C^2RoPE’s effectiveness. The code is be available at https://github.com/ErikZ719/C2RoPE.

C²RoPE (Causal Continuous Rotary Positional Encoding)

Core Idea: A spatially-aware extension of RoPE that preserves 2D visual continuity and models causal relationships via Chebyshev distance, preventing locality loss and long-term attention decay in vision-language models.

Problem It Solves: Standard RoPE applies 1D temporal indices to 2D image patches, breaking spatial continuity across columns and causing progressive neglect of earlier visual tokens due to temporal-only causal assumptions; this harms 3D scene reasoning where spatial relationships are critical.

How It Works:

  1. Hybrid Positional Index Construction: Combine 1D temporal position t with 2D Cartesian spatial coordinates (x, y) to form a triplet (t, x, y) for each image token
  2. Frequency Allocation Strategy: Partition RoPE’s frequency bands across the three positional components (temporal + 2D spatial) to encode spatio-temporal information in the rotary embeddings
  3. Chebyshev Causal Masking: Replace temporal distance with Chebyshev distance max(|x₁-x₂|, |y₁-y₂|) in 2D space to determine causal dependencies, preserving spatial locality while maintaining autoregressive properties

Architecture:

flowchart LR
    A[Image Patches] --> B[Triplet Index: t,x,y]
    B --> C[Frequency-Partitioned RoPE]
    C --> D[Spatial Embeddings]
    E[Token Pairs] --> F[Chebyshev Distance]
    F --> G[Causal Mask]
    D --> H[Attention + Mask]
    G --> H
    H --> I[Spatially-Aware Features]

Where It’s Useful:

  • 3D visual question answering and scene reasoning tasks requiring spatial understanding
  • Multi-image or video understanding where column-wise continuity matters (e.g., video frames as grid)
  • Any vision-language model suffering from spatial locality loss in long visual sequences

Limitations / Caveats:

  • Requires modifying the base model’s positional encoding scheme (not a drop-in replacement)
  • Chebyshev distance assumes 2D grid structure; unclear how to extend to irregular 3D point clouds or non-grid visual representations
  • Frequency allocation strategy adds hyperparameter choices (how to split frequencies across t, x, y dimensions)

Internal Use Cases:

  • Multimodal RecSys: When ranking items represented as 2D grids (e.g., product image galleries, fashion layouts), C²RoPE could preserve spatial relationships between items better than 1D positional encoding
  • Document Understanding: For ranking or extracting information from documents with 2D layout (tables, forms), spatial continuity matters more than pure temporal order
  • Video Recommendation: Frame-level features arranged in grids could benefit from spatial positional encoding to capture shot composition, not just temporal sequence
