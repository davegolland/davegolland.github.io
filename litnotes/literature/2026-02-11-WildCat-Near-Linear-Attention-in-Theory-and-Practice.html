WildCat: Near-Linear Attention in Theory and Practice

Source: arXiv:cs.LG Published: 2026-02-10 Rank: #3 (score: 0.98, max_sim: 0.83) Reading time: ~0 min Tags: Uncategorized

Abstract

We introduce WildCat, a high-accuracy, low-cost approach to compressing the attention mechanism in neural networks. While attention is a staple of modern network architectures, it is also notoriously expensive to deploy due to resource requirements that scale quadratically with the input sequence length . WildCat avoids these quadratic costs by only attending over a small weighted coreset. Crucially, we select the coreset using a fast but spectrally-accurate subsampling algorithm — randomly pivoted Cholesky — and weight the elements optimally to minimise reconstruction error. Remarkably, given bounded inputs, WildCat approximates exact attention with super-polynomial error decay while running in near-linear time. In contrast, prior practical approximations either lack error guarantees or require quadratic runtime to guarantee such high fidelity. We couple this advance with a GPU-optimized PyTorch implementation and a suite of benchmark experiments demonstrating the benefits of WildCat for image generation, image classification, and language model KV cache compression.

Key Techniques

  • Weighted coreset selection
  • randomly pivoted Cholesky
  • optimal weighting for reconstruction error minimization

WildCat (Weighted Coreset Attention via Randomly Pivoted Cholesky)

Core Idea: Approximates full attention by attending only over a small weighted coreset selected via randomly pivoted Cholesky decomposition, achieving super-polynomial error decay with near-linear runtime complexity.

Problem It Solves: Existing attention approximations either lack provable error guarantees or require quadratic runtime to achieve high fidelity; WildCat provides both theoretical guarantees ( error) and practical efficiency ( time).

How It Works:

  1. Coreset Selection: Apply randomly pivoted Cholesky decomposition to the kernel matrix (derived from query-key similarities) to select a small subset of representative tokens, where the algorithm greedily selects tokens that maximize remaining spectral information.
  2. Optimal Weighting: Compute importance weights for each selected coreset token to minimize reconstruction error when approximating the full attention output, ensuring the weighted subset preserves critical spectral properties of the original attention matrix.
  3. Sparse Attention Computation: Perform attention only between queries and the weighted coreset of keys/values, reducing computational cost from to where for sublinear coreset size.

    B --> C[Coreset Selection m << n]
    C --> D[Optimal Weight Computation]
    D --> E[Weighted Sparse Attention]
    E --> F[Output ≈ Full Attention]

Where It’s Useful:

  • Long-context language modeling with KV cache compression for reduced memory footprint during inference
  • High-resolution image generation and classification where spatial tokens create quadratic bottlenecks
  • Any Transformer-based architecture requiring attention over sequences with tokens

Limitations / Caveats:

  • Requires bounded input assumption for theoretical guarantees (e.g., normalized queries/keys)
  • Coreset selection adds preprocessing overhead proportional to , which may dominate for very small sequence lengths
  • Optimal weighting computation requires solving a linear system, adding numerical stability considerations
  • Effectiveness depends on spectral structure of attention matrices (works best when attention has low effective rank)

Internal Use Cases:

  • RecSys candidate ranking: Compress cross-attention between user history (long sequence) and candidate items, enabling real-time inference over thousands of historical interactions
  • LLM serving: Apply to KV cache compression in multi-turn conversations, reducing memory bandwidth and latency for long-context retrieval-augmented generation
  • Document ranking: Efficient cross-attention between query tokens and large document collections without materialized quadratic similarity matrices
