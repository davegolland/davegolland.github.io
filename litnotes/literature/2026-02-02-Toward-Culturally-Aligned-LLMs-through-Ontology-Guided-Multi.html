Toward Culturally Aligned LLMs through Ontology-Guided Multi-Agent Reasoning

Source: arXiv:cs.IR Published: 2026-01-29 Rank: #8 (score: 0.98, max_sim: 0.83) Reading time: ~15 min Tags: LLM Code: https://arxiv.org/pdf/2601.21700v2

Abstract

Large Language Models (LLMs) increasingly support culturally sensitive decision making, yet often exhibit misalignment due to skewed pretraining data and the absence of structured value representations. Existing methods can steer outputs, but often lack demographic grounding and treat values as independent, unstructured signals, reducing consistency and interpretability. We propose OG-MAR, an Ontology-Guided Multi-Agent Reasoning framework. OG-MAR summarizes respondent-specific values from the World Values Survey (WVS) and constructs a global cultural ontology by eliciting relations over a fixed taxonomy via competency questions. At inference time, it retrieves ontology-consistent relations and demographically similar profiles to instantiate multiple value-persona agents, whose outputs are synthesized by a judgment agent that enforces ontology consistency and demographic proximity. Experiments on regional social-survey benchmarks across four LLM backbones show that OG-MAR improves cultural alignment and robustness over competitive baselines, while producing more transparent reasoning traces.

Key Techniques

  • Ontology-Guided Multi-Agent Reasoning
  • World Values Survey value summarization
  • value-persona agents with judgment synthesis
  • cultural ontology construction via competency questions

Value-Persona Agent Instantiation and Ontology-Guided Synthesis

Core Idea: Instantiate multiple value-persona agents conditioned on demographically similar profiles and ontology-consistent value relations, then synthesize outputs through a judgment agent that enforces consistency.

Problem It Solves: A single LLM output may not reflect nuanced cultural values, and outputs may contradict the constructed cultural ontology. This method grounds reasoning in specific personas while enforcing global ontology consistency.

How It Works:

  1. At inference time, given an input query and a target demographic profile d, retrieve the k most demographically similar profiles from WVS using cosine similarity over demographic embeddings
  2. For each similar profile, retrieve ontology-consistent value relations R_i from the cultural ontology (filtering to relations relevant to the query domain)
  3. Construct a value-persona prompt that includes: the input query, demographic profile d, and the retrieved ontology relations R_i
  4. Instantiate n independent value-persona agents by prompting the LLM with distinct persona prompts conditioned on different subsets of similar profiles
  5. Feed all n persona outputs to a judgment agent that:
    • Checks consistency against the ontology: for each pair of outputs, verify they don’t violate ontology relations
    • Computes demographic proximity scores: weight outputs by how closely their implied values match the target profile d
    • Synthesizes a final output by selecting or merging persona outputs that maximize ontology consistency and demographic alignment
  6. Return the synthesized output with reasoning trace

Architecture:

flowchart LR
    A["Query + Target Profile d"] --> B["Retrieve Similar Profiles"]
    B --> C["Retrieve Ontology Relations R_i"]
    C --> D["Construct Persona Prompts"]
    D --> E["Instantiate Value-Persona Agents"]
    E --> F["Collect Persona Outputs"]
    F --> G["Judgment Agent:<br/>Check Ontology Consistency<br/>Compute Demographic Proximity"]
    G --> H["Synthesize Final Output"]

Where It’s Useful:

  • Culturally-aware chatbots and decision support systems
  • Applications requiring transparent reasoning aligned with user demographics and values
  • Multi-stakeholder decision making where consistency is critical

Limitations / Caveats:

  • Persona instantiation may increase inference cost due to multiple forward passes
  • Judgment agent synthesis heuristics may not optimally resolve conflicting persona outputs
  • Demographic proximity assumes available demographic metadata and similarity metrics
  • Ontology consistency checking is post-hoc and may not catch all violations

Internal Use Cases:

  • Could apply to recommendation ranking by instantiating “user-value agents” that reason about item recommendations based on inferred user values and ontology constraints
  • Similar multi-agent synthesis could be used in ranking systems where different rankers need to be reconciled based on consistency criteria
