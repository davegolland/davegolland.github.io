Price of universality in vector quantization is at most 0.11 bit

Price of universality in vector quantization is at most 0.11 bit

Source: arXiv:cs.LG Published: 2026-02-05 Rank: #10 (score: 0.49, max_sim: 0.81) Reading time: ~0 min Tags: Uncategorized

Abstract

Fast computation of a matrix product is a workhorse of modern LLMs. To make their deployment more efficient, a popular approach is that of using a low-precision approximation in place of true (“weight-only quantization”). Information theory demonstrates that an optimal algorithm for reducing precision of depends on the (second order) statistics of and requires a careful alignment of vector quantization codebook with PCA directions of (a process known as “waterfilling allocation”). Dependence of the codebook on statistics of , however, is highly impractical. This paper proves that there exist a universal codebook that is simultaneously near-optimal for all possible statistics of , in the sense of being at least as good as an -adapted waterfilling codebook with rate reduced by 0.11 bit per dimension. Such universal codebook would be an ideal candidate for the low-precision storage format, a topic of active modern research, but alas the existence proof is non-constructive. Equivalently, our result shows existence of a net in that is a nearly-optimal covering of a sphere simultaneously with respect to all Hilbert norms.

Universal Vector Quantization Codebook

Core Idea: A single vector quantization codebook can achieve near-optimal rate-distortion performance across all possible input statistics simultaneously, sacrificing at most 0.11 bits per dimension compared to an input-adapted waterfilling codebook.

Problem It Solves: Traditional optimal vector quantization requires aligning the codebook with the principal component analysis (PCA) directions of input data X, making the codebook dependent on X’s statistics—this is impractical for deployment scenarios like LLM weight quantization where diverse input distributions are encountered.

How It Works:

  1. Input: Weight matrix W ∈ ℝⁿ to be quantized, unknown/varying input statistics for X
  2. Universal Codebook Construction: Construct a codebook that forms a near-optimal covering of a unit sphere simultaneously with respect to all Hilbert norms (existence proven non-constructively)
  3. Quantization: Map each vector in W to nearest codeword using the universal codebook, independent of X statistics
  4. Output: Quantized weight approximation Ŵ that achieves rate within 0.11 bits/dimension of the optimal X-adapted waterfilling allocation

Where It’s Useful:

  • Weight-only quantization for LLM deployment (INT4/INT8 storage formats)
  • Universal compression codecs that must handle diverse data distributions
  • Hardware-friendly quantization schemes with fixed codebooks burned into chips
  • Embedding table compression in recommendation systems with varying query distributions

Limitations / Caveats:

  • The existence proof is non-constructive—no explicit algorithm provided to build the universal codebook
  • The 0.11 bit penalty is an upper bound; practical construction methods may have larger gaps
  • Theorem applies to asymptotic regime; finite-dimensional performance may vary
  • Does not address computational cost of codebook search during encoding

Internal Use Cases:

  • Deploy a single quantized model variant for LLM inference that performs near-optimally regardless of downstream task/prompt distribution
  • Universal embedding compression for recommendation systems where item/user feature distributions shift over time
  • Cross-domain transfer learning with quantized models (train on one domain, deploy on another without re-quantizing)
