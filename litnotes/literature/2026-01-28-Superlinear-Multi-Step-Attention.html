Superlinear Multi-Step Attention

Source: arXiv:cs.LG Published: 2026-01-26 Rank: #11 (score: 0.93, max_sim: 0.86) Reading time: ~20 min Tags: RAG, Transformer, Search, Block-Sparse, NSA

Abstract

In this paper, we propose \textbf{Superlinear attention}, a fully trainable multi-step attention architecture that achieves subquadratic complexity for long sequences while preserving \textbf{random context access} (a.k.a.\ structural non-exclusion): no eligible token position is structurally excluded from being selected for attention. Superlinear attention reformulates standard causal self-attention as a multi-step search problem with steps, yielding an overall complexity of . To illustrate the architecture, we present a baseline implementation, which is algorithmically analogous to standard jump search. In this instantiation, the first step performs span-search to select relevant spans of the sequence, and the second step applies span-attention (standard attention restricted to the selected spans). In an upscaled configuration for robustness, we achieve an average decoding throughput of 114 tokens/sec at 1M context length and 80 tokens/sec at 10M context in our implementation on a modified 30B hybrid MoE model on a single B200 GPU. With limited training, we also obtain strong performance on the NIAH (Needle In A Haystack) task up to 256K context length, demonstrating that the routed span selection is learnable end-to-end. This paper emphasizes architectural formulation, scaling analysis, and systems feasibility, and presents initial validation; comprehensive quality evaluations across diverse long-context tasks are left to future work.

Analysis

Extending Sparse Attention Architecture: Superlinear attention builds on the foundational concept of Block Sparse Attention by introducing a learnable, hierarchical selection mechanism rather than relying on fixed or heuristically-determined block masks. Where Block Sparse Attention partitions sequences into static blocks and applies hand-crafted or simple learned gating to select which blocks interact, Superlinear attention reformulates the problem as a multi-step search procedure: the first step performs span-search across the entire sequence (O(L^{3/2}) for N=2), then the second step applies refined attention only within selected spans. This shifts from static block patterns to dynamically routed span selection, preserving the efficiency gains of block-wise computation while enabling end-to-end training of the selection mechanism itself. The paper demonstrates this is actually learnable on NIAH tasks up to 256K context, suggesting that span routing learns meaningful semantic boundaries rather than relying on fixed windowing rules.

Bridging Learned Selection and Hardware Efficiency: The paper’s approach shares the dual goals of Native Sparse Attention (NSA)—combining compression and selection—but inverts the order of abstraction. NSA compresses token blocks into summaries, then selects relevant blocks for detailed attention. Superlinear attention instead performs coarse-grained span search first (selecting which regions deserve fine-grained processing), then applies standard attention within those regions. This parallels NSA’s two-stage pipeline but with explicit complexity analysis: by formulating attention as an N-step search problem yielding O(L^{1+1/N}) complexity, the paper provides a principled framework for trading off search granularity against computational cost. Unlike Fixed-pattern Sparse Attention, which locks sparsity patterns into the model architecture, Superlinear attention makes sparsity learnable, suggesting that optimal context routing can be discovered during training rather than predetermined.

Novel Contribution: Subquadratic Complexity with Random Context Access: The key innovation is preserving random context access—ensuring no eligible token is structurally excluded—while achieving subquadratic complexity. This directly challenges the implicit trade-off in prior sparse attention methods: fixed patterns preserve random access but sacrifice learning; learned selection often implies structural exclusions (tokens above a threshold are never attended to). Superlinear attention avoids this by treating sparse selection as a learned ranking/search problem where all positions remain theoretically reachable, but efficiently in practice. The 30B hybrid MoE results (114 tokens/sec at 1M context on a B200 GPU) demonstrate systems-level feasibility, suggesting that the theoretical complexity improvements translate to real throughput gains—a gap that often exists between sparse attention theory and practice.
